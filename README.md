# Sistema de Gerenciamento da UFC – Campus e Cursos (CRUD em Python)

Este projeto implementa um sistema simples de gerenciamento de Campi e Cursos da UFC utilizando Programação Orientada a Objetos (POO) e um menu interativo no terminal.
O objetivo é permitir que o usuário cadastre, visualize, edite e remova campi e cursos usando listas e classes em Python.

### O que o programa faz?

O programa cria uma estrutura composta por:

Classe Curso: representa um curso com um nome.

Classe Campus: representa um campus que possui vários cursos.

Classe UFC: armazena a lista de todos os campi cadastrados.

O usuário navega pelo sistema por meio de um menu textual e pode realizar operações de CRUD tanto para campi quanto para cursos.

### Funcionalidades
- Gerenciamento de Campi

O usuário pode:

Cadastrar um novo campus

Listar todos os campi cadastrados

Editar o nome de um campus

Remover um campus existente

- Gerenciamento de Cursos (para um campus específico)

Dentro de um campus selecionado, o usuário pode:

Adicionar um curso

Listar todos os cursos do campus

Editar o nome de um curso

Remover um curso

### Menu Interativo

O programa apresenta um menu principal com as opções:

1 - Cadastrar Campus
2 - Listar Campi
3 - Editar Campus
4 - Remover Campus
5 - Gerenciar Cursos
0 - Sair


Ao selecionar "Gerenciar Cursos", outro menu é exibido para manipular os cursos de um campus específico.
